Explorando las Propiedades de las Tablas con CSS:

La propiedad borders se utiliza para definir el estilo de los bordes de la tabla.

La propiedad size establece el tamaño de los elementos de la tabla, como el ancho de las celdas y la altura de las filas.

La propiedad alignment determina la alineación del contenido dentro de las celdas de la tabla, como el texto y otros elementos.
