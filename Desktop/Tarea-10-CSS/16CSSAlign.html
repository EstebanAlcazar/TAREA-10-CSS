Explorando la propiedad CSS Align:

La propiedad text-align se utiliza para alinear el texto y otros elementos en un contenedor. Puede tener los siguientes valores:

  • left: Alinea el contenido a la izquierda.
  • right: Alinea el contenido a la derecha.
  • justify: Justifica el contenido, creando espacios adicionales entre las palabras para que el texto se alinee en ambos lados.
  • center: Centra el contenido horizontalmente en el contenedor.
