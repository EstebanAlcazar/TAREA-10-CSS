Contenido con Opacity

Explorando la propiedad CSS Opacity:

La propiedad opacity se utiliza para controlar la transparencia de un elemento. Tiene un valor comprendido entre 0 (totalmente transparente) y 1 (totalmente opaco).

En esta página, al pasar el cursor sobre el contenido, la opacidad aumenta de 0.8 a 1, lo que hace que el elemento se vuelva completamente opaco.
