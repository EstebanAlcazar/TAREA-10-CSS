Modern Text Styling

Experimenta con diferentes estilos de texto para mejorar la apariencia de tu sitio web.

Explorando las Propiedades de Texto:

La propiedad color define el color del texto.

La propiedad text-align alinea el texto dentro de su contenedor.

La propiedad text-decoration añade decoraciones al texto, como subrayado o línea a través.

La propiedad text-transform controla la transformación del texto, como mayúsculas o minúsculas.

La propiedad letter-spacing ajusta el espaciado entre las letras del texto.
