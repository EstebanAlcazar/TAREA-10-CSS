Explorando la propiedad CSS Layout - display: inline-block:

La propiedad display: inline-block permite que los elementos se muestren como bloques, pero fluyen como elementos en línea, lo que significa que pueden tener un ancho y alto definidos y pueden estar alineados horizontalmente.

En esta página, los elementos de tipo "caja" se muestran en línea uno al lado del otro, pero tienen un ancho y alto definidos, lo que les permite tener un diseño de cuadrícula.

Además, se ha agregado una transición suave al hacer hover sobre las cajas para agregar interactividad y mejorar la experiencia del usuario.
